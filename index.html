Q1: 技術独自性

Yes: 特許・独自技術・技術的優位性あり
No: 汎用技術・競合多数・技術的差別化なし

Q2: 顧客独自性

Yes: 特定分野・ニッチ市場・独自顧客基盤
No: 一般的顧客・広範囲対象・差別化なし

Q3: 価格プレミアム

Yes: 高付加価値・プレミアム価格設定可能
No: 価格競争・コモディティ化・価格優位性なし

Q4: 利益率優位

Yes: 高利益率・効率的運営・収益性優位
No: 低利益率・コスト圧迫・収益性劣位

Q5: 独占可能性

Yes: 市場独占・寡占・参入障壁高
No: 激しい競争・参入容易・独占性なし

Q6: 業界依存度

Yes: 特定業界依存・業界動向影響大
No: 複数業界・業界横断・依存度低

Q7: 設計組込

Yes: 顧客製品に組込・切替困難・長期契約
No: 表面的使用・切替容易・短期契約

Q8: 収益規模

Yes: 大規模市場・高収益ポテンシャル
No: 小規模市場・限定的収益機会

Q9: 成長継続

Yes: 持続的成長・拡張性・将来性高
No: 成長停滞・縮小傾向・将来性低

📖 本調査について

調査の目的

本調査は、LPWA/IoT市場において、企業の収益性を左右する要因を明らかにすることを目的としています。同じ技術市場に参入していても、企業によって収益性に大きな差が生まれるのはなぜでしょうか。本調査では、「事業の立地」—すなわち、どの技術規格を選び、どのような顧客に、どのような用途で価値を提供するか—という選択が、収益性を決定づけるという視点から検証を行います。

ご協力のお願い

皆様には、25社のLPWA/IoT関連企業について、9つの観点から評価をお願いします。ご協力いただいたデータは、技術市場における事業立地の選択と収益性の関係を理解するための貴重な資料となります。どうぞよろしくお願いいたします。

📋 評価項目の詳細定義（全9項目）

📌 評価の進め方：以下の9つの質問それぞれについて、各企業のLPWA事業が「Yes基準（1点）」に該当するか「No基準（0点）」に該当するかを判定してください。判断が困難な場合は「N/A」を選択してください。

質問1：独自技術もしくは特許による参入障壁が確認できますか？

✓ Yes基準：
独自IP・特許網や代替困難なコア技術を持つ
例：SEP、Technical IP、Core competance
✗ No基準：
汎用技術のみ使用、技術的差別化なし
例：汎用的なチップ・モジュールのみ採用

質問2：特定用途・業界に特化し、強固な顧客基盤を持っていますか？

✓ Yes基準：
特定用途（検針・見守り等）で明確な「勝ちパターン」を確立
例：スマートメーター専業、農業IoT特化
✗ No基準：
「IoT全般対応」の全方位展開
例：「あらゆる用途に使えます」という汎用訴求

質問3：価格競争を回避し、高単価を維持できていますか？

✓ Yes基準：
サービス込み高単価契約、顧客の価格受容性が高い
例：技術サポート・保守込みの長期契約
✗ No基準：
価格競争が主軸、薄利多売モデル
例：モジュール単価競争に巻き込まれている

質問4：シェア拡大より利益率維持を優先していますか？

✓ Yes基準：
ニッチ市場で高収益率を維持
例：水道検針等の特定用途に絞り込み
✗ No基準：
トップライン成長第一主義
例：スマートシティ全般など大規模市場参入志向

質問5：競合が少ない独占・寡占可能な立地を重視していますか？

✓ Yes基準：
競合1-2社の事実上独占状態
例：規格立ち上げメンバーとして先行者利益確保
✗ No基準：
多数競合のシェア争い市場
例：標準化により参入障壁が低い汎用チップ市場

質問6：顧客にとってミッションクリティカルな用途ですか？

✓ Yes基準：
インフラ監視・防災等、停止時損失が甚大
例：ガス漏れ検知、電力網監視、防災システム
✗ No基準：
「便利」程度の位置づけ、エンタメ・ガジェット
例：在庫管理の効率化、エンターテインメント用途

質問7：顧客製品にDesign-inされ、切り替え困難ですか？

✓ Yes基準：
専用基板・独自プロトコル、長期ロックイン
例：顧客製品の標準仕様として採用済み
✗ No基準：
互換モジュール、短期契約中心
例：ピン互換の代替品が複数存在

質問8：新規参入が困難な構造的障壁がありますか？

✓ Yes基準：
複雑な許認可、過去数年新規参入なし
例：専門資格・認証必須、過去3年の新規参入2社以下
✗ No基準：
参入容易、異業種参入多数
例：認証済モジュール利用で容易に参入可能

質問9：個別対応中心で、大企業が規模で攻めにくいですか？

✓ Yes基準：
現場調整・すり合わせ必須、多品種少量対応
例：顧客ごとのカスタマイズ・現地調整が必須
✗ No基準：
標準化・大量生産可能、規模の経済が効く
例：完全標準化製品で大量バラマキ戦略が可能

17. SJI / SJIT (🇰🇷)

簡単な年表()

時期フェーズ象徴的な出来事・製品動き
2016-2018参入・黎明期Sigfox Monarch PartnershipフランスSigfox社と戦略的パートナーシップを締結し、世界初のSigfox Monarch（グローバルローミング対応）モジュール「SFMシリーズ」を開発・量産化。物流トラッキング分野でのLPWA活用を開拓。
2019-2021マルチモード展開期Sigfox + GPS/Wi-Fi Hybrid Module単一通信の弱点を補うため、Sigfox通信とGPS/Wi-Fi測位を組み合わせたハイブリッドモジュールを展開。物流パレットや資産追跡向けに低消費電力かつ広域通信可能なソリューションを確立。
2022-2024分社・再編期SJIT Spin-off / LSM SeriesSigfox/LoRaデュアル対応モジュール（LSMシリーズ）を拡販する中、2023年末にハードウェア製造・LPWA事業を新設法人SJIT（SeongJi Industrial Technology）へ移管し、IoT専業体制へ転換。

代表的なLPWA製品例

  • LSM100A / LSM110A (Dual Stack Module): SigfoxとLoRaWANのデュアルスタックに対応したハイブリッドLPWAモジュール。1つのハードウェアで複数のLPWA規格をカバーし、グローバル展開するIoT機器に採用。
