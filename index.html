📖 本調査について

調査の目的

本調査は、LPWA/IoT市場において、企業の収益性を左右する要因を明らかにすることを目的としています。同じ技術市場に参入していても、企業によって収益性に大きな差が生まれるのはなぜでしょうか。本調査では、「事業の立地」—すなわち、どの技術規格を選び、どのような顧客に、どのような用途で価値を提供するか—という選択が、収益性を決定づけるという視点から検証を行います。

LPWA市場の特徴

LPWA（Low Power Wide Area）は、IoT向けの無線通信技術です。この市場には、LoRaWAN、NB-IoT、LTE-M、Wi-SUNなど、10種類以上の技術規格が併存しており、それぞれ異なる用途や地域で使われています。また、IoT機器は一度導入されると10〜20年使い続けられるため、途中で技術を変更することが難しく、企業は最初の立地選択に大きく制約されます。

なぜ「事業立地」が重要なのか

実務では「成長市場だから参入しよう」「市場規模が小さいから撤退しよう」といった、市場全体の成長性や規模を基準とした判断がよく行われます。しかし、戦略の要は、売り先と売り物を絞り込み、標準メニューから微妙にシフトさせたうえで、両者の新しい組み合わせを指向することにあります。利益率の水準は事業の立地で決まり、市場全体が成長していても、自社の立ち位置が適切でなければ収益は上がりません。

本調査では、「どの技術で、どのような顧客に、どのように価値を提供するか」という事業立地の選択が、収益性にどう影響するのかを実証的に検証します。

📋 評価項目の詳細定義（全9項目）

📌 評価の進め方：以下の9つの質問それぞれについて、各企業のLPWA事業が「Yes基準（1点）」に該当するか「No基準（0点）」に該当するかを判定してください。判断が困難な場合は「N/A」を選択してください。

質問1：独自技術もしくは特許による参入障壁が確認できますか？

✓ Yes基準：
独自IP・特許網や代替困難なコア技術を持つ
例：SEP、Technical IP、Core competance
✗ No基準：
汎用技術のみ使用、技術的差別化なし
例：汎用的なチップ・モジュールのみ採用

質問2：特定用途・業界に特化し、強固な顧客基盤を持っていますか？

✓ Yes基準：
特定用途（検針・見守り等）で明確な「勝ちパターン」を確立
例：スマートメーター専業、農業IoT特化
✗ No基準：
「IoT全般対応」の全方位展開
例：「あらゆる用途に使えます」という汎用訴求

質問3：価格競争を回避し、高単価を維持できていますか？

✓ Yes基準：
サービス込み高単価契約、顧客の価格受容性が高い
例：技術サポート・保守込みの長期契約
✗ No基準：
価格競争が主軸、薄利多売モデル
例：モジュール単価競争に巻き込まれている

質問4：シェア拡大より利益率維持を優先していますか？

✓ Yes基準：
ニッチ市場で高収益率を維持
例：水道検針等の特定用途に絞り込み
✗ No基準：
トップライン成長第一主義
例：スマートシティ全般など大規模市場参入志向

質問5：競合が少ない独占・寡占可能な立地を重視していますか？

✓ Yes基準：
競合1-2社の事実上独占状態
例：規格立ち上げメンバーとして先行者利益確保
✗ No基準：
多数競合のシェア争い市場
例：標準化により参入障壁が低い汎用チップ市場

質問6：顧客にとってミッションクリティカルな用途ですか？

✓ Yes基準：
インフラ監視・防災等、停止時損失が甚大
例：ガス漏れ検知、電力網監視、防災システム
✗ No基準：
「便利」程度の位置づけ、エンタメ・ガジェット
例：在庫管理の効率化、エンターテインメント用途

質問7：顧客製品にDesign-inされ、切り替え困難ですか？

✓ Yes基準：
専用基板・独自プロトコル、長期ロックイン
例：顧客製品の標準仕様として採用済み
✗ No基準：
互換モジュール、短期契約中心
例：ピン互換の代替品が複数存在

質問8：新規参入が困難な構造的障壁がありますか？

✓ Yes基準：
複雑な許認可、過去数年新規参入なし
例：専門資格・認証必須、過去3年の新規参入2社以下
✗ No基準：
参入容易、異業種参入多数
例：認証済モジュール利用で容易に参入可能

質問9：個別対応中心で、大企業が規模で攻めにくいですか？

✓ Yes基準：
現場調整・すり合わせ必須、多品種少量対応
例：顧客ごとのカスタマイズ・現地調整が必須
✗ No基準：
標準化・大量生産可能、規模の経済が効く
例：完全標準化製品で大量バラマキ戦略が可能
